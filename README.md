# 🚀 Jenkins CI Project (Maven)


Set up Continuous Integration using Jenkins to automatically build a Maven project.

## 🛠️ Tech Stack

* Jenkins
* Maven
* GitHub
* Java

## ⚙️ Steps Performed

1. Installed Jenkins on Ubuntu
2. Configured Jenkins and plugins
3. Created Maven project
4. Pushed code to GitHub
5. Connected Jenkins with GitHub repo
6. Configured Maven build
7. Triggered build successfully

## ▶️ Build Command

```
clean install
```
